## Setup Instructions

1. **Local Development**:
   - Clone or download the repository
   - Open `index.html` in your browser
   - Alternatively, use the provided build tools:
     - Windows: Run `run.bat` and select option 1 to open the website
     - Unix/Linux/Mac: Run `bash run.sh` and select option 1 to open the website
     - With Make: Use `make open` if you have GNU Make installed
     - With Node.js: Use `npm start` if you have Node.js installed

2. **Development Tools**:
   - Install development dependencies: `npm install`
   - Validate HTML: `npm run validate:html` or `make validate-html`
   - Validate CSS: `npm run validate:css` or `make validate-css`
   - Validate JavaScript: `npm run validate:js` or `make validate-js`
   - Validate all: `npm run validate` or `make validate`

3. **Deployment**:
   - Use the provided build tools to create a deployment package:
     - Windows: Run `run.bat` and select option 2 to create a ZIP package
     - Unix/Linux/Mac: Run `bash run.sh` and select option 2 to create a ZIP package
     - With Make: Use `make package` if you have GNU Make installed
     - With Node.js: Use `npm run package` if you have Node.js installed
   - Upload the generated files or ZIP package to your web hosting service
   - No compilation required as this is a static website
